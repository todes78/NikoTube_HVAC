# Преобразователь частоты для индукционного нагрева

В состав преобразователя частоты входят:
* преобразователь частоты
* система охлаждения

В проект входит:
* программа графической панели системы управления преоьразователем частоты
* программа системы управления системы охлаждения
* программа графической панели системы управления системы охлаждения

## Система управления системой охлаждения

Система управления системой охлаждения состоит:

* Программируемый логический контроллер **M241** (Schneider Electric)

    Состав ПЛК:
    1. Центральный процессор M241CE40T
    1. Модуль аналогового ввода TM3AI8
        * Давление на входе насосов (внутренний контур)
        * Давление на входе теплообменника (внутренний контур)
        * Давление на выходе теплообменника (внутренний контур)
        * Давление на входе теплообменника (внешний контур)
        * Давление на выходе теплообменника (внешний контур)
        * Контроль проводимости воды (кондуктометр ACM-1E24000)    
        
    1. Модуль аналогового ввода термодатчиков TM3TI4
        * Температура на входе теплообменника (Pt100, внутренний контур)
        * Температура на выходе теплообменника (Pt100, внутренний контур)
        * Температура на входе теплообменника (Pt100, внешний контур)
        * Температура на выходе теплообменника (Pt100, внешний контур)
        
Программное обеспечение SoMachine v4.2 

* Графическая панель **MT8050iE** (Weintek)
Программное обеспечение EasyBuilder Pro v6.03.02.294 Build 2019.12.03
